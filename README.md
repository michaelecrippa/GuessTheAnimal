# GuessTheAnimal
## How to run the project ?
 - Fetch/Clone the project  
 - Install dependencies ( ghc, cabal, stack ) 
 - Run `stack exec GuessTheAnimal-exe`  
## How to test the project? 
 - Run `stack test`
